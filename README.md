### Project Overview  
This project explores ride-share pricing data to support dynamic pricing strategies. It focuses on identifying key cost drivers and simulating pricing models that enhance revenue, customer experience, and operational efficiency.

---

### Progress Summary  
- Performed data cleaning, outlier detection, and exploratory analysis  
- Built automated pipelines for data ingestion and KPI tracking  
- Engineered features using encoding and grouping techniques  
- Designed rule-based pricing models using time and inventory signals  
- Simulated revenue lift and visualized pricing comparisons
